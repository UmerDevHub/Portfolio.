## 🔍 FILE NAME VERIFICATION

Make sure file names are EXACTLY like this:
```
✅ CORRECT:
- pocket-journal.jpg
- carpool-app.jpg
- heart-predictor.jpg

❌ WRONG (Won't Work):
- Pocket Journal.jpg        (spaces)
- pocket_journal.jpg        (underscore)
- pocketjournal.jpg         (no hyphen)
- pocket-journal.JPG        (uppercase)
- pocket-journal.png        (wrong extension)
```
