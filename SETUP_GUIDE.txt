### How to Add Images

1. **Get images** (your own screenshots, Unsplash, Pexels, etc.)
2. **Save to** `assets/images/` folder
3. **Update HTML** - Find image URLs and replace:

```html
<!-- Find this in projects.html: -->
<img alt="Project Name" 
     src="https://lh3.googleusercontent.com/..." 
     class="..."/>

<!-- Replace with: -->
<img alt="Project Name" 
     src="assets/images/project-name.jpg" 
     class="..."/>
```

## 🎨 Customization Tips

### Change Rotating Text
Open `index.html` and find:
```javascript
const rotatingTexts = [
    'I am Umer Nisar',
    'Flutter Developer',
    'Full-Stack Engineer',
    // Add more here
];
```

### Change Resume File
In `index.html`, find "Download Resume" button section:
```javascript
if(button.textContent.includes('Download Resume')) {
    button.addEventListener('click', () => {
        link.href = 'assets/resume/resume.html'; // Change this path
```

### Change Animation Speed
In `index.html` style section:
```css
@keyframes fadeInOut {
    /* Animation takes 4 seconds per cycle */
    /* Change 4s to any value you want */
}
```

And JavaScript:
```javascript
setInterval(() => {
    // Change 2000 (milliseconds = 2 seconds) to another value
}, 2000);
```

## 🐛 Troubleshooting

### Resume doesn't download
- **Problem**: File not found at `assets/resume/resume.html`
- **Solution**: Create the folder and file
- **Check**: File path in button click handler

### Text animation not working
- **Problem**: Element with id="rotating-text" not found
- **Solution**: Verify the ID in index.html matches JavaScript

### Form not submitting
- **Problem**: Form has empty fields or invalid email
- **Solution**: Fill all fields, use valid email format

### Projects grid looks wrong
- **Problem**: Missing closing `</div>` tags
- **Solution**: Already fixed! But check console for errors

### Images not showing
- **Problem**: Wrong image path or file doesn't exist
- **Solution**: 
  - Check file exists in `assets/images/`
  - Verify path in HTML matches exactly
  - Check file name capitalization

## 📧 Contact Form - Email Integration (Optional)

Currently form saves data locally. To send emails:

### Option 1: EmailJS (Recommended)
1. Go to https://www.emailjs.com/
2. Sign up free
3. Add this to contact.html before `</head>`:
```html
<script type="text/javascript" src="https://cdn.jsdelivr.net/npm/@emailjs/browser@3/dist/index.min.js"></script>
<script>
  emailjs.init('YOUR_PUBLIC_KEY'); // Get from EmailJS
</script>
```

### Option 2: Formspree
1. Go to https://formspree.io/
2. Create account
3. Change form action:
```html
<form action="https://formspree.io/f/YOUR_FORM_ID" method="POST">
```
